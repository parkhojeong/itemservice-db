* @EventListener(ApplicationReadyEvent.class)
- focus on application ready
- completely after spring container init including spring aop
- @PostConstruct,
  - focus on bean ready
  - start before spring aop registry(aop registry is after bean ready)
  - caution: with AOP proxy. ex, @Transactional

* profile
- spring read profile from spring.profiles.active=
- register bean by @Profile(value) 
- default profile : "default"
- test config: /src/test/resources/application.properties
