* @Repository
- @Repository class is target of Exception translation AOP
  - register PersistenceExceptionTranslator
  - translate Jpa Exception to Spring DataAccessException

* flow
1. EntityManager throw PersistenceException
2. JpaItemRepositoryV1 throw PersistenceException
3. AOP Proxy translate PersistenceException to DataAccessException
4. Service handle DataAccessException

* spring boot 
- auto register PersistenceExceptionTranslationPostProcessor
- Advisor make @Repository AOP Proxy

* translation code
- EntityManagerFactoryUtils.convertJpaAccessExceptionIfPossible()
